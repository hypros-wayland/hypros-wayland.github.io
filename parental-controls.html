Parental controls

Child profiles with PIN locks, screen limits, and filtering.

Overview

PlasmaTV includes a full parental control system designed for families. The adult user (tv-user) can create up to 16 child profiles with individual restrictions.

Child accounts

Each child gets a Linux user account (child-user-0 through child-user-15) with:

  • Restricted shell (rbash) — a locked-down bash that only allows whitelisted commands
  • Locked PATH — only /usr/local/lib/plasmatv-child-bin/ is in the path, containing only electron
  • No sudo access — no password, no privilege escalation
  • Screen time tracking — usage is logged to /var/lib/plasmatv/screentime/

PINs

Child profiles can be protected with a numeric PIN (salted SHA-256). The PIN is verified by a root helper (plasmatv-verify-pin) callable only from the user-selector session via a tightly scoped sudoers rule.

Screen time limits

The plasmatv-timelimit-daemon runs as a systemd service that polls every 60 seconds. When a child exceeds their daily limit, the Screen Time Blocker Electron app launches in fullscreen with no easy escape (Escape and Alt+F4 are prevented via e.preventDefault()).

DNS filtering

The plasmatv-dns-rules service uses iptables NAT rules to redirect all DNS traffic to Cloudflare for Families (1.1.1.3), which blocks malware and adult content at the DNS level. IPv6 DNS is also redirected.

Managing children

Use the Child Manager app from the Bigscreen launcher. It provides a GUI for:

  • Creating new child profiles
  • Setting or changing PINs
  • Deleting child accounts
  • Viewing screen time usage

Sudoers scoping

Access to the parental control system is carefully scoped through multiple sudoers files:

Rule fileWhat it allows
plasmatv-adulttv-user can manage children, set PINs, factory reset
plasmatv-selectorselector-user can only verify PINs and start sessions
plasmatv-user-unitsAll users can restart SDDM (for return-to-selector flow)
tv-user-session-switchtv-user can switch between Bigscreen and Desktop
